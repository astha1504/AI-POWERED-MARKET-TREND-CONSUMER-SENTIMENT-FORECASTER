# AI-POWERED MARKET TREND & CONSUMER SENTIMENT FORECASTER

## Frontend Module

### Features
- Authentication (Register/Login)
- Dashboard with product analytics
- Product detail with:
  - Rating distribution
  - Sentiment analysis
  - Market trend chart
- Compare products with AI recommendation
- Watchlist functionality
- CSV-based scraped dataset integration

### Tech Stack
- React (Vite)
- Tailwind CSS
- Recharts
- PapaParse
- LocalStorage Authentication

Developed as part of Internship Team Project.
